# json-pretty-print-online
`json-pretty-print-online` is a simple React app for prettifying JSON text. Simply paste your JSON text [here](https://samceracas.github.io/json-pretty-print-online/) and copy the output.

# Building and Running  
To run server side rendering for this app, simply use:  
```
yarn ssr
```

For development:
```
yarn start
```

For building a static site:
```
yarn build
```

`gh-pages` deployment:
```
yarn deploy
```
